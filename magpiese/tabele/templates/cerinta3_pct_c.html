Cerinta

Afisarea rezultatului unei cereri care extrage informatii din cel putin 3 tabele si le filtreaza cu ajutorul a cel putin 2 contitii.

Comanda SQL

SELECT nr_inmatriculare, tabele_comenzi.id AS 'id comanda', prenume

FROM magpiese.tabele_masini, magpiese.tabele_comenzi, magpiese.tabele_clienti

WHERE prenume = 'Bogdan' && tabele_comenzi.id BETWEEN 2 AND 7 && tabele_comenzi.masina_nr_id = tabele_masini.vin && tabele_comenzi.client_nume_id = tabele_clienti.cnp;

Rezolvare
